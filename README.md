# Marketing Funnel & Conversion Performance Analysis

## Objective
Analyze customer behavior and marketing campaign performance using Power BI.

## Dataset
Bank Marketing Dataset

## Tools Used
- Power BI

## Key Insights
- Total customers analyzed: 11,162
- Cellular contact channel performed best
- Management and blue-collar segments had highest customer counts
- Age group 30-39 showed highest representation

## Recommendations
- Focus on cellular campaigns
- Target high-performing customer segments
- Improve low-performing channels
- Use age-based marketing strategies
